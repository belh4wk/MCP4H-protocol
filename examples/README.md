# MCP4H™ Examples

- `messages/` Canonical maintained JSON examples
- `arduino/` Hardware demos
- `experimental/` Non-conformant or work-in-progress demos

## Mirror rule

Some legacy example paths are intentionally preserved for backward compatibility.
Do not treat those mirrors as source-of-truth files.

Canonical source: `examples/messages/`

To resync legacy mirrors:

```bash
python tools/sync_examples.py
```
